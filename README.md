Intelligent Recommendation System

Overview

This project utilizes machine learning to build a content-based recommendation system that suggests products based on user input.

---

Features
- Content-based filtering using TF-IDF
- Cosine similarity for recommendation scoring
- Intelligent matching based on product category and description
- Filters low-similarity recommendations for better accuracy

---

️ Tech Stack
- Python
- Pandas
- Scikit-learn

---

How to Run

pip install pandas scikit-learn

python Main. py
